
# شركة جسر البيانات للحلول التقنية — موقع هبوط (RTL)

موقع ثابت (Static) خفيف بدون أُطر ثقيلة. يدعم العربية (RTL) وتصميم سريع الاستجابة.

## المحتويات
- `index.html`
- `assets/css/styles.css`
- `assets/js/app.js`
- `assets/img/logo.svg`

## التخصيص السريع
- غيّر الألوان والخطوط من :root داخل `assets/css/styles.css`.
- حدّث روابط واتساب والبريد داخل `index.html` (القسم: Contact).
- أضف أكواد التحليلات (GA4/Meta Pixel) في `<head>` حسب تعليقات المكان المخصص.

## النشر
- **Vercel**: اربط المستودع واضغط Deploy (مشروع Static).
- **Netlify**: اسحب وأفلت المجلد، أو اربط Git (لا يحتاج Build).
- **Shared Hosting**: ارفع الملفات كما هي إلى مجلد الموقع (public_html).

## النماذج
حاليًا النموذج يفتح البريد المحلي عبر `mailto:`. لالتقاط العملاء تلقائيًا:
- استخدم خدمات مثل Formspree/Formspark أو أربط فورم بـ **Odoo**/**Zoho**/**HubSpot**.
- استبدل event الإرسال في `app.js` بطلب `fetch` لنقطة نهاية مناسبة.

## Agent Mode (اختياري لاحقًا)
يمكن إضافة مساعد تشخيصي صغير يقرأ FAQ وملفات الخدمات لاقتراح باقة مناسبة وجمع بيانات العميل. مفضل بعد الإطلاق.
